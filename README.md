# water-consumpsion-monitoring-system
Water Level Monitoring System is designed using Arduino Uno and sensors to monitor the water level in storage tanks. The system provides a real-time alert using a buzzer when the tank is full, preventing overflow and water wastage. It offers a reliable and cost-effective solution for efficient water management.

# Tech Stack
  Hardware. Arduino Uno, Water Level Sensor, Buzzer (Alert System), Jumper Wires, Breadboard
  Python, Html, CSS, Js
  
 # Working Explanation
- Water level sensor detects the level of water in the tank  
- Sensor sends signals to Arduino Uno  
- Arduino processes the input data  
- When water reaches the maximum level, Arduino activates the buzzer  
- Buzzer gives an alert indicating the tank is full  

 # Key Points
- Simple and cost-effective system  
- Prevents water overflow  
- Provides real-time alert  
- Easy to install and maintain
